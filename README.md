### Setup detailes  
▪ **Window Manager** ➙ DWM  
▪ **Terminal** ➙ Alacritty  
▪ **Compositor** ➙ Picom  
▪ **File manager** ➙ Dolphin  
▪ **Browser** ➙ Vivaldi  
▪ **Notification daemon** ➙ Dunst  
▪ **Widgets & Bar** ➙ Eww  
▪ **Menu** ➙ Rofi  
▪ **Editor** ➙ Visual studio code  
▪ **Music player** ➙ Ncmpcpp
